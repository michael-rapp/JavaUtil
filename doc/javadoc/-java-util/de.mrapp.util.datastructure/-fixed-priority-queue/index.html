FixedPriorityQueue

class FixedPriorityQueue<T> : PriorityQueue<T>

An extension of the class PriorityQueue that has a fixed size. If adding an item causes the maximum size to be exceeded, the smallest/first element is removed.

Parameters

T - The type of the items that are contained by the queue

maxSize - The maximum size of the queue
