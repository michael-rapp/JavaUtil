@JvmOverloads fun removeSuffix(file: File, separator: String = SUFFIX_SEPARATOR, firstOccurrence: Boolean = true): File

Returns a specific file with its suffix removed. If the file does not have a suffix, the name is returned as it is. If firstOccurrence is true, the suffix is considered to start at the first occurrence of the separator, otherwise it is considered to start at the last occurrence.
