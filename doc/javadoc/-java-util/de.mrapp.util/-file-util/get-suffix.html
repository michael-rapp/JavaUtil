getSuffix

@JvmOverloads fun getSuffix(file: File, separator: String = SUFFIX_SEPARATOR, firstOccurrence: Boolean = true): String?

Returns the suffix of a specific file or null, if the file does not have a suffix. If firstOccurrence is true, the suffix is considered to start at the first occurrence of the separator, otherwise it is considered to start at the last occurrence. The separator is not included in the returned suffix.

@JvmOverloads fun getSuffix(fileName: String, separator: String = SUFFIX_SEPARATOR, firstOccurrence: Boolean = true): String?

Returns the suffix of a file with a specific fileName or null, if the file does not have a suffix. If firstOccurrence is true, the suffix is considered to start at the first occurrence of the separator, otherwise it is considered to start at the last occurrence. The separator is not included in the returned suffix.
