getUniqueFile

@JvmOverloads fun getUniqueFile(file: File, separator: String = SUFFIX_SEPARATOR, suffix: String? = null): File

Returns a variant of a specific file which is unique within its parent directory. If the name of the given file is not unique, a number will be appended. Optionally, a suffix may be added to the name of the given file.
