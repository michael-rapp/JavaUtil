getTruncatedName

fun getTruncatedName(clazz: Class<*>): String

Returns a truncated version of a specific class' full qualified name. For example, if the full qualified name of the class is 'com.xyz.Abc', the result will be 'c.x.Abc'.

Parameters

clazz - The class, whose full qualified name should be turned into a truncated version

Return
The truncated version of the given class' full qualified name
